利用規約

最終更新日: 2026年4月2日

第1条（本サービスの目的）

本利用規約（以下「本規約」といいます。）は、StudySnap運営者（以下「運営者」といいます。）が提供する学習記録管理アプリケーション「StudySnap」（以下「本サービス」といいます。）について、ユーザーが利用する際に適用されます。

本サービスは、ユーザーが日々の学習を写真やテキストで記録し、グループ機能を通じて他のユーザーと学習状況を共有することで、学習習慣の定着を支援することを目的としています。

2. 本サービスにつき、運営者が別途定めるプライバシーポリシー、ガイドラインその他の規約（以下「個別規約」といいます。）も、本規約の一部を構成するものとします。本規約の定めと個別規約の定めが異なる場合、当該個別規約の定めが優先して適用されるものとします。

3. 運営者は、随時、運営者の判断により、本規約を任意に変更または改定することができます。ユーザーは、変更または改定後の本規約を確認のうえ、本サービスをご利用ください。ユーザーが変更後に本サービスを利用する場合、変更後の本規約に同意したものとみなします。

4. ユーザーが未成年者の場合、親権者または未成年後見人の同意を得たうえで、本サービスをご利用ください。未成年者のユーザーが本サービスを利用した場合、親権者等の同意を得たものとみなします。

2. 会員登録手続の際には、所定の入力フォームに必要事項を正確に入力してください。ユーザーは、入力された事項が事実と異ならないことを表明および保証するものとします。

3. 運営者は、ユーザーが過去に本規約に違反したことがある場合、本規約に違反するおそれがある場合その他本サービスの運営上支障があると判断した場合には、会員資格を付与せず、一旦付与した場合にもこれを取り消す場合があります。

4. 会員登録手続は、会員となるユーザーご本人が行ってください。代理による登録は一切認められません。

5. ユーザーは、会員としての地位およびサービスの利用により運営者に対して取得した一切の権利を譲渡、転貸、担保差入その他形態を問わず処分することはできません。

第4条（本サービスの利用）

1. 会員登録したユーザーは、アプリにログインして、本サービスを利用できます。ログイン方法は以下のとおりです。

  • メールアドレスおよびパスワードでログインする方法
  • Googleアカウントでログインする方法
  • Apple IDでログインする方法

また、不正利用防止のため、一部の機能（承認機能等）のご利用には電話番号認証が必要となる場合があります。

2. 本サービスにおいて、ユーザーに対して提供するサービスの内容は、随時変更されることがあります。

3. 本サービスに関する問い合わせその他ユーザーから運営者に対する連絡または通知、および本規約の変更に関する通知その他運営者からユーザーに対する連絡または通知は、運営者の定める方法で行うものとします。

1. 本サービスの一部の機能（以下「有料サービス」といいます。）は、別途定める料金をお支払いいただくことでご利用いただけます。

3. 有料サービスの解約は、Apple IDの設定画面から、ユーザー自身で行うものとします。解約後も、契約期間の満了まで有料サービスをご利用いただけます。

4. 有料サービスの料金は、理由の如何を問わず、返金いたしません。ただし、法令に別段の定めがある場合はこの限りではありません。

第7条（グループ機能）

1. グループの管理者は、メンバーの承認・拒否・除外の権限を有します。

2. グループ内での相互承認機能は、勉強記録の信頼性を高めるためのものであり、承認の正確性についてはユーザー間の責任となります。

3. グループ内で投稿されたコンテンツについても、本規約の各条項が適用されます。

4. グループ内のチャット機能では、運営者が用意した定型文メッセージを送信できます。チャットの内容はグループメンバー全員に公開されます。

5. グループメンバーの学習レポート（アプリ内の勉強時間、アプリ外の勉強時間、および合計の勉強時間を含む）は、同一グループのメンバー間で閲覧可能です。

第7条の2（アプリ外の勉強記録）

1. ユーザーは、本アプリ外で行った勉強の記録（教科、時間等）を手動で入力することができます。

2. アプリ外の勉強記録の正確性については、入力したユーザー自身の責任とし、運営者はその内容の真実性について一切保証しません。

3. アプリ外の勉強記録は、アプリ内の記録と区別して表示されます。

第7条の3（ブロック機能）

1. ユーザーは、他のユーザーをブロックすることができます。ブロックした場合、当該ブロックされたユーザーの投稿およびチャットメッセージがブロックしたユーザーの画面上で非表示となります。

2. ブロックはブロックを行ったユーザー個人に対してのみ適用され、グループ内の他のメンバーには影響しません。

3. ブロックされたユーザーには、ブロックされたことは通知されません。

4. ユーザーはいつでもブロックを解除することができます。

  14. 個人が特定される連絡先を本サービス内に投稿する行為
  15. 運営者が事前に許諾しない宣伝、広告、勧誘または営業行為
